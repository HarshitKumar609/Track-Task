# 🚀 Task Tracker Backend API

A production-ready REST API built for a Task Tracker application using Node.js, Express.js, and MongoDB.

---

## 📌 Features

- Create Task
- Get All Tasks
- Get Task By ID
- Update Task
- Delete Task
- MongoDB Integration
- Clean MVC-like structure (Controller, Service, Model)
- Error Handling
- RESTful API Design

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
- cors
